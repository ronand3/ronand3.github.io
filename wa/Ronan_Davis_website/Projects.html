My Projects

More Then What Meets The Eyes

For this project, I used Adobe Photoshop and Adobe Premiere to edit and organize the visual content. I worked with video footage that I collected around Boulder to capture different parts of the town. The goal of the project was to show two different sides of life in Boulder through visuals and video.

Video: More Than What Meets The Eyes

Popping Bubbles

For this project, I used JavaScript to code a simple interactive game using the p5 library. The game gives the player a set number of lives to complete each round. It also includes a live scoreboard that keeps track of the player’s progress and records the number of rounds completed.

Project Notes

Both of these projects pushed me to work with several new applications and coding languages that I had no previous experience using. While it was challenging at first, learning these tools helped me develop new technical skills and a better understanding of digital design and development. By the end of the process, I was able to apply what I learned to create a variety of different projects and build confidence working with new technologies.
